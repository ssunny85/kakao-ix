# Kakao IX 과제

## Project 구동

Backend, Frontend 서버 구동후 Front 개발서버로 접속하여 과제 확인 가능합니다.

### Mongodb install

```shell script
$ brew tap mongodb/brew
$ brew install mongodb-community
```

### Backend 구동

```shell script
$ cd backend
$ npm install
$ npm serve
```

### Front Vue 구동

```shell script
$ cd todo-vue
$ npm install
$ npm serve
```

### Front React 구동

```shell script
$ cd todo-react
$ npm install
$ npm serve
```
